# GoEnba - 農家パートナー募集LP

GoEnbaの農家向けランディングページ

## 概要

農家の伴走支援型・産直ECプラットフォーム「GoEnba」の農家パートナー募集用ランディングページです。

## 特徴

- 静的HTML（単一ファイル）
- レスポンシブデザイン
- フェードインアニメーション
- ハンバーガーメニュー

## 使用技術

- HTML5
- CSS3
- JavaScript（Vanilla）
- Google Fonts

## ローカルで開く

ブラウザで `farmer-landing.html` を開くだけで表示できます。
